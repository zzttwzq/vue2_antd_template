<h1 align="center">vue_antd_template</h1>

## 浏览器支持
现代浏览器及 IE10

## 使用
### yarn
```bash
$ yarn install
$ yarn dev
```
### or npm
```
$ npm install
$ npm run dev
```
## 项目介绍

├── bootstrap.js 配置路由，请求拦截等

│   │   └── userAccount 账号管理
│   │   └── menu 菜单管理

│       ├── projectNamelist 名单列表
│       │   ├── button_list_view.vue
